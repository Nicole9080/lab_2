## Описание реализации

**Структуры данных:**
* **Граф** представлен двумя способами:
  * матрица смежности (квадратная матрица $V \times V$);
  * матрица инцидентности (прямоугольная матрица $V \times E$).
* **BST** — бинарное дерево поиска с узлами, содержащими значение и ссылки на левое и правое поддеревья.
* **Куча** для Heap Sort — массив, поддерживающий свойство max‑heap.

**Сложность алгоритмов:**
* Построение матриц: $O(V^2)$ для смежности, $O(V \cdot E)$ для инцидентности.
* Поиск компонент связности (DFS): $O(V + E)$.
* Вставка/поиск/удаление в BST: $O(h)$, где $h$ — высота дерева (в среднем $O(\log V)$, в худшем $O(V)$).
* Heap Sort: $O(n \log n)$ во всех случаях.
